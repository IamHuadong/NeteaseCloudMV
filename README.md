# NeteaseCloudMV
## 网易云MV在线交流
## 技术栈
- webpack
- Vue
## clone运行必知
- 上传了webpack配置，运行命令参考本项目文件的webpack.json里的scripts项
- 天气组件需要您在小白开放接口注册会员，得到会员id才能成功请求url。
- 网易云MV接口需要运行本地node server，具体参考 https://binaryify.github.io/NeteaseCloudMusicApi/#/
